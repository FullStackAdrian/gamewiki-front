# Zelda BOTW Compendium App

Este proyecto es una aplicación web de ejemplo que simula un compendio de elementos del juego "The Legend of Zelda: Breath of the Wild". Permite explorar y gestionar información sobre monstruos y materiales, ofreciendo una interfaz para visualizar, añadir, editar y eliminar elementos.

La aplicación está construida usando React y sigue principios de arquitectura limpia para una mejor organización y escalabilidad.
