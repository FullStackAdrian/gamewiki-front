# Dependencias del Proyecto

Este proyecto utiliza las siguientes tecnologías y librerías clave:

## Frontend:

*   **React (v18+)**: Una librería de JavaScript para construir interfaces de usuario. Proporciona un modelo declarativo y basado en componentes para el desarrollo.
*   **React Router (v6+)**: Una colección de componentes de navegación para React, esencial para manejar el enrutamiento dentro de la aplicación.
*   **Tailwind CSS**: Un framework CSS de primera utilidad ("utility-first") para construir rápidamente UIs personalizadas directamente en tu marcado. Se utiliza para el estilizado de todos los componentes.
*   **TypeScript**: Un superconjunto tipado de JavaScript que añade tipos estáticos opcionales. Mejora la legibilidad, mantenibilidad y ayuda a prevenir errores en tiempo de desarrollo.

## Otras herramientas y librerías (ejemplos, pueden variar):

*   **Vite**: Herramienta de construcción para configurar rápidamente un entorno de desarrollo React. (Depende de cómo se inicializó el proyecto).
*   **Axios**: Para realizar peticiones HTTP a APIs externas.
*   **ESLint**: Herramientas para mantener la consistencia del código y aplicar buenas prácticas de estilo.
*   **Zod**: (Opcional) Librerías para validación de esquemas (e.g., formularios, datos de API).

Estas dependencias proporcionan un ecosistema robusto y moderno para el desarrollo de aplicaciones web, centrándose en la eficiencia del desarrollador, la escalabilidad y una experiencia de usuario fluida.
